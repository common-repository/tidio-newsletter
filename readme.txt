== Description ==
                            
Have a look at Tidio Elements Newsletter – an advanced newsletter system designed to meet the requirements of even most demanding users – no limit for emails, tracking sent emails and managing the address list to name just a few of its functions!

[youtube http://www.youtube.com/watch?v=CMaHRB8Vk0M]
                            
Installation and configuration take no more than a couple of minutes and get you set to easily manage your address list from the level of WordPress. Sounds interesting? It's only just the start!
                                    
= Selected functionalities: =
        
* Address list manager (no emails limit) 
* WYSIWYG new email creator 
* Tracking sent emails (shows which recipients have opened the email) 
* Email campaign manager 
* Support for website without a configured STMP server 
* In-built PopUp and SignUp form 
* Export of data to CSV 
* Free!

== Frequently Asked Questions ==

= Where are the e-mail addresses stored?  =
All the e-maill addresses are stored in the Tidio Elements database.

= Can I export the e-mail addresses?  =
You can export all the e-mail addresses at any given time.

= Does the plugin support various languages?   =
Yes, you can edit the content displayed to users with no problems.

= How will the e-mail addresses be sent?   =
There are three options to choose from: through the Tidio Elements server, through the SMTP server, through your WordPress website.

= What if my website doesn’t support SMTP?   =
Then the e-mails can be sent through the Tidio Elements API or the SMTP server.
